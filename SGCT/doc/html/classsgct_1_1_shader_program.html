Detailed Description

Helper class for handling compiling, linking and using shader programs. Current implementation only supports vertex and fragment shader. Uniform and attribute handling must be managed explicitly but it is possible to poll the Shader program for uniform and attribute locations.

Default only sets the program name.Shaders objects won't be created until the any shader source code is set. The program will be created when the createAndLink() function is called. Make sure the shader sources are set before calling it.

Parameters

sgct::ShaderProgram::~ShaderProgram ( void  )

The destructor clears the shader data vector but the program can still be used. The program have to be destroyed explicitly by calling deleteProgram. This is so that programs can be copied when storing in containers.
